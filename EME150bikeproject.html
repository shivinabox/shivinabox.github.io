Bike Projects

DISCLAIMER: The project provided on this webpage has been used in Graded Assignments already. Copying any part of this assignment will constitue to Plagiarism !

The main goal of this project was to reverse analysis of a bike to find critical components of

  • Creating a dimensional diagram of the bike and relevant dimensions.

  • Performing load envelope analysis and determining maximum loads in a bikes life cycle

  • Finding critical components in all load envelopes

  • Analysing the critical components for material properties and cost analysis

  • Evalutaing the design of critical components against mechanical failure and determining thier lives based on the components design

  • Use the analysis done so far to create a life and maintenance schedule for the bicycle

  • Suggest design modifications to improve the life of the bike

The Report consists of the following topics

  • Abstract

  • Introduction

  • Methodology

  • Load Envelopes Considered and their Proper Analyses

  • Analysis of Critical Components

  • Suggested Design Modifications

  • Conclusions and Recommendations
